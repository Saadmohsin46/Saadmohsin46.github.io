Vim Basics

In this lesson, we will cover the basic commands and usage of vim. Vim is a highly configurable text editor built to enable efficient text editing. It is an improved version of the vi editor distributed with most UNIX systems.

Getting Started with Vim

  • Open a terminal.
  • Type vim followed by the filename to open or create a file, e.g., 
    vim myfile.txt
    .

    • Basic Commands

    • i: Enter insert mode (start editing the file).
    • Esc: Exit insert mode (return to command mode).
    • :w: Save the file.
    • :q: Quit vim.
    • :wq: Save and quit vim.
    • :q!: Quit without saving.

    Editing Text

    To make text changes, first ensure you are in command mode (press Esc), then use the following commands:

    
        dd - Delete the current line
        yy - Copy (yank) the current line
        p - Paste the copied line
        u - Undo the last action

    Useful Tips

    • Use Ctrl + r to redo the last undone action.
    • Use v to enter visual mode, allowing you to select text.
    • Use : to enter command-line mode and run advanced commands.
